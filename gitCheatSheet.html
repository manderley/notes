Git cheat sheet

Last updated: 4 October 2015

Frequently used commands

  • Create a new branch, switch to it, and set upstream configuration (you could also create an alias to automatically set the upstream branch): 
    $ git checkout -b <branch-name>
$ git push -u origin <branch-name>
  • Integrate changes from one branch to another using rebase: 
    $ git checkout <branch-without-changes>
$ git rebase <branch-with-changes>
