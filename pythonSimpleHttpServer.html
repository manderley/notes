Python's SimpleHTTPServer

Last updated: 20 October 2015

How to use it

Python's built-in web server is really simple, requiring no installation or configuration, only that Python is installed.

To use, in the directory containing the files for your website, type: 

$ python -m SimpleHTTPServer

You should then see: 

Serving HTTP on 0.0.0.0 port 8000 ...

8000 is the default port. To use a different one: 

$ python -m SimpleHTTPServer [port]

Once the server is running you should be able to view your site in your browser at http://127.0.0.1:[port]. If there is no index.html file in the root directory then all files and directories will be listed.

For more information see the Python documentation.
